Tips for Writing Clean Code

Three rules that cover most cases:

  1. Name things clearly — a good name is better than a comment.
  2. Keep functions small — if it needs a scrollbar, split it.
  3. Delete unused code — version control has your back.

Simple, but easy to forget under pressure.
